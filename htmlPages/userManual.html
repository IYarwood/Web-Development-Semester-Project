User Manual

On this page, you can find the overview of the website, summary of pages on this website, and the descriptions of pages on this website.

Overview

This website allows students to register for courses through a simple web form. It also allows students to drop a course on a separate form. Users can also enroll students, add instructors, and add courses. In future updates, all of these changes will be saved to a database, allowing for easy updates to the college system. In addition, users will have the option to view who is registered for certain courses. All of these pages and functionalities can be seen listed above in the navigation bar, which will take you to whatever page you click.

Student Enrollment

At the top of this webpage you will first see the page title/header, followed by our navigation bar underneath of it. You can click on the text in the navigation bar to jump to different pages on the website. At the very bottom of the page, you will see our page footer, with our web project title and contributors.
On the main part of the page (in the middle) you will see the web form. The form will be structured like this:

First Name
The first name of the student enrolling
Required
Spaces allowed
Examples include "John", "Lee John", "Mark", etc.
Last Name
The last name of the student enrolling
Required
Spaces allowed
Examples include "Smith", "McStein", "Knowles-Carter", etc.
Year
The year of the student to be enrolled
Required
Options are Freshman, Sophomore, Junior, or Senior
Student Major
The enrolling student's college major, fully typed
Not required
Spaces allowed
Examples include "Computer Science", "Math", "Computer Science, Physics", etc.
Email Address
The enrolling student's preferred email address we may contact
Must follow the format abc123@marietta.edu. The first three characters must be letters. The next three characters must be numbers. The email address must end with @marietta.edu
Examples include "lok006@marietta.edu" etc.

Add Course

At the top of this webpage you will first see the page title/header, followed by our navigation bar underneath of it. You can click on the text in the navigation bar to jump to different pages on the website. At the very bottom of the page, you will see our page footer, with our web project title and contributors.
On the main part of the page (in the middle) you will see the web form. The form will be structured like this:

Semester
The semester the class to be added takes part in
Required
Pick an option
Options include "Fall", "Spring", and "Summer"
Year
The year the class to be added takes part in
Required
4-digit numbers only
Examples include "2025", "2028", etc.
Course Prefix
The course to be added's department prefix
Required
3 or 4 capital letters
Examples include "CSCI", "PSYC", "MATH", etc.
Course Number
The course to added's course number
Required
3 digit numbers only, not larger than 499
Examples include "101", "330", "215", etc.
Course Section
The course to added's section number
Required
Two-digit numbers only
Exmaples include "01", "02", "03", etc.
Course Name
The course's name
Examples include "Computer Programming II", "Introduction to Philosophy", etc.
Room
The course's meeting location (building and room)
Not Required
Must be formatted as a building name and room number. The building name will be a string that can have spaces. The room number can be 2 or 3 digits, and it may end with a letter
Examples include "Selby 240", "Hermann Fine Arts Center 103a", etc.
Days Offered
The course's day schedule options
Pick an option
Options include "Monday, Wednesday, Friday", "Monday, Wednesday", "Tuesday, Thursday",
"Monday", "Tuesday", "Wednesday", and "Thursday"
Time
The course's start-end time
Pick an option after Days Offered
Valid times:
Monday, Wednesday, Friday
  • 8:00AM to 8:50AM
  • 9:00AM to 9:50AM
  • 10:00AM to 10:50AM
  • 11:00AM to 11:50AM
  • 12:00PM to 12:50PM
  • 1:00PM to 1:50PM
  • 2:00PM to 2:50PM
  • 3:00PM to 3:50PM
Monday, Wednesday
  • 9:00AM to 10:50AM
  • 1:00PM to 2:15PM
  • 1:00PM to 2:50PM
  • 2:00PM to 3:15PM
Tuesday, Thursday
  • 8:00AM to 9:15AM
  • 9:30AM to 10:45AM
  • 11:00AM to 12:15PM
  • 1:00PM to 2:15PM
  • 2:30PM to 3:45PM
Any Single Day
  • 1:00PM to 4:00PM
  • 2:00PM to 5:00PM
  • 7:00PM to 9:30PM
Credit Hours
The course's credit hour value
Required
Must be a number 0-4
Examples include "3", "1", and "2"
Instructor First Name
The course's instructor's First name
Required
Spaces allowed
Examples include "John", "Lee John", "Mark", etc.
Instructor Last Name
The course's instructor's last name
Required
Spaces allowed
Examples include "Smith", "McStein", "Knowles Carter", etc.
Enrollment Cap
The course's maximum number of students
Required
Two digit numbers only
Examples include "20", "50", "10", etc.
